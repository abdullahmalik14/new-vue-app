# public/config

All `assetMap*.json` files here are **generated** from `src/config/` (global + optional `assetMap.<section>.json`).

Edit files under `src/config/` only. Run `npm run sync:asset-map` or start the dev server to refresh copies.

**URL model (environments, inheritance, relative vs CDN paths):** see `src/config/assetMap.README.md`.
